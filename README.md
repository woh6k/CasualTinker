# CasualTinker (休闲匠魂)

![Version](https://img.shields.io/badge/Minecraft-1.19.2-green.svg) ![Forge](https://img.shields.io/badge/Forge-Required-orange.svg) ![Tinkers' Construct](https://img.shields.io/badge/Addon-Tinkers'_Construct_3-blue.svg)

**休闲匠魂 (CasualTinker)** 是一个为 Minecraft 1.19.2 打造的匠魂 3 (Tinkers' Construct 3) 附属模组。

本模组尚未开发完成，是新人的第一次尝试。

本模组融合了各种奇奇怪怪的灵感，为匠魂系统增加了全新的材料、强力的特性。
可以用粘液球和书合成书，与匠魂百科结合看



**安装前置**: 确保安装了**1.19.2对应forge、Tinkers' Construct 3 和 地幔前置**。
